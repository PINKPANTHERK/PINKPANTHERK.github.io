Jupter Notebook&MarkDown

#Jupter Notebook的使用#
快捷键的使用（Jupter顶部菜单使用Help > Keyboard Shortcuts）
1.执行当前cell，并自动跳到下一个cell：Shift Enter
2.执行当前cell，执行后不自动调转到下一个cell：Ctrl-Enter
3.是当前的cell进入编辑模式：Enter
4.退出当前cell的编辑模式：Esc
5.删除当前的cell：双D%load test.py #test.py是当前路径下的一个python文件
6.为当前的cell加入line number：L
7.将当前的cell转化为具有一级标题的maskdown：1
8.将当前的cell转化为具有二级标题的maskdown：2
9.将当前的cell转化为具有三级标题的maskdown：3
10.将当前的cell转化为具有三级标题的maskdown：Y
11.为一行或者多行添加/取消注释：Crtl /
12.撤销对某个cell的删除：z
13.浏览器的各个Tab之间切换：Crtl PgUp和Crtl PgDn

单元格操作
1.删除单元格：选择该单元格，然后依次点击Edit -> Delete Cell
2.移动单元格：依次点击Edit -> Move cell [up | down]
3.剪贴某个单元格：先点击Edit -> Cut Cell，然后在点击Edit -> Paste Cell [Above | Below]
4.合并单元格：点击Edit -> Merge Cell [Above | below]

##Jupyter Notebook如何导入代码
1.在需要导入该段代码的cell中输入: <%load .py #.py是当前路径下的一个python文件>
2.运行该cell，可利用快捷键“Shift + Enter”
3.运行后%load **.py被自动加入了注释符号#，test.py中的所有代码都被load到了当前的cell中
